# ❓ Quiz App

A simple and interactive **Quiz Application** built using modern web technologies.
Users can answer multiple-choice questions and see their score at the end.

---

## 🚀 Live Demo

(https://quiz-app-shruti-chourasiya.netlify.app/)


## 📌 Features

* 🧠 Multiple choice quiz questions
* ⏭️ Next question navigation
* 📊 Score calculation
* ⚡ Fast and responsive UI
* 🎨 Clean and modern design

## 🛠️ Technologies Used

* HTML
* CSS
